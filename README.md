# Akshit Verma's Portfolio

A modern, responsive personal portfolio website showcasing my projects, skills, and experience. Built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scroll navigation
- Animated sections with Intersection Observer
- Contact form with email integration
- Project showcase with hover effects
- Skills and education sections
- Modern minimalist design with accent color scheme

## Technologies Used

- HTML5
- CSS3 (Custom variables, Flexbox, Grid)
- Vanilla JavaScript
- Intersection Observer API

## Setup

1. Clone the repository:
```bash
git clone https://github.com/Akshit9150/portfolio.git
```

2. Open the project:
```bash
cd portfolio
```

3. Open `index.html` in your browser to view the website.

## Project Structure

```
portfolio/
├── index.html      # Main HTML file
├── styles.css      # CSS styles
├── script.js       # JavaScript functionality
└── README.md      # Project documentation
```
